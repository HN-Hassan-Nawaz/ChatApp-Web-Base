Step 1 : Create a new react project using vite js
         npm create vite@latest (write here your project name)

Step 2 : cd (project-name)
         Install npm
         npm install

Step 3 : Install Tailwind Css
         npm install -D tailwindcss postcss autoprefixer
         npx tailwindcss init -p

         Configure your template paths
         content:[
         "./index.html",
         "./src/**/*.{js,ts,jsx,tsx}",
         ],

Add Tailwind directives to your CSS (in index.css file)

         @tailwind base;
         @tailwind components;
         @tailwind utilities;

Start Project/Run project on browser
         npm run dev


install react router dom for routing:
         npm i react-router-dom
